# identity-platform Plugin — Developer Guidance

## Skills in this Plugin

- `skills/asgardeo-auth/` — Asgardeo CLI org setup, OAuth2 app registration, SDK install, and minimal login/logout for React, Next.js, Vue, and Express

## Key Reference Files

- `skills/asgardeo-auth/schema/config-profile.yaml` — schema for `.asgardeo/config-<profile>.yaml` (read before generating or updating any config file)
- `skills/asgardeo-auth/references/react.md` — `@asgardeo/react` integration pattern
- `skills/asgardeo-auth/references/nextjs.md` — `@asgardeo/nextjs` (≥15.3) integration pattern
- `skills/asgardeo-auth/references/vue.md` — `@asgardeo/vue` integration pattern
- `skills/asgardeo-auth/references/express.md` — `@asgardeo/express` integration pattern
- `skills/asgardeo-auth/bin/asgardeo` — bundled Asgardeo CLI binary (macOS arm64); copied to `/usr/local/bin/asgardeo` on first use
